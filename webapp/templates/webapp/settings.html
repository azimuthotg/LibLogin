Help & Documentation
MikroTik Integration

To integrate with MikroTik Hotspot, add this JavaScript to your login.html:

<script>
fetch('{{ request.scheme }}://{{ request.get_host }}/api/login-background/?hotspot_name=YOUR_ID')
  .then(response => response.json())
  .then(data => {
    if (data.success) {
      document.body.style.backgroundImage =
        `url($${data.imageUrl})`;
    }
  });
</script>
Quick Tips
  • Use router-specific backgrounds for different locations
  • Recommended image size: 1920x1080 pixels
  • Images are automatically optimized on upload
  • Only one background can be active per router at a time
  • Use the API endpoint in your MikroTik configuration
Security
  • Change default admin password
  • Only authorized users can manage images
  • API endpoint is CORS-enabled for MikroTik
  • Regular backups recommended
