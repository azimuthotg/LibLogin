คู่มือ MikroTik API Integration
วิธีเชื่อมต่อ MikroTik Hotspot กับระบบ LibLogin
1
เพิ่ม Hotspot ในระบบ
ไปที่หน้า จัดการ Hotspot แล้วเพิ่ม Hotspot พร้อมตั้งชื่อ hotspot_name ให้ตรงกับ Router ของคุณ
2
อัปโหลด Background Image
ไปที่หน้า จัดการรูปภาพ แล้วอัปโหลดภาพและตั้ง Status เป็น Active สำหรับ Hotspot ที่ต้องการ
3
แก้ไข login.html ใน MikroTik
เพิ่ม JavaScript ด้านล่างนี้ใน /flash/hotspot/login.html ก่อนแท็ก </body>
<!-- LibLogin Background Loader --> <script> fetch('{{ request.scheme }}://{{ request.get_host }}{% url 'login-background' %}?hotspot_name=YOUR_HOTSPOT_ID') .then(r => r.json()) .then(data => { if (data.success) { document.body.style.backgroundImage = `url(${data.imageUrl})`; document.body.style.backgroundSize = 'cover'; document.body.style.backgroundPosition = 'center'; } }); </script>
4
แทนที่ YOUR_HOTSPOT_ID
เปลี่ยน YOUR_HOTSPOT_ID เป็น hotspot_name ที่ตั้งไว้ในระบบ เช่น hotspot_lib
API Parameters
Parameter ประเภท คำอธิบาย
hotspot_name Optional ชื่อ Hotspot ที่ต้องการดึง Background ถ้าไม่ระบุจะใช้ค่า Default
Response Fields
Field ประเภท คำอธิบาย
success boolean true = มีภาพ Active อยู่
imageUrl string URL ของรูปภาพ (full URL)
hotspot_name string Hotspot ที่ใช้งาน (อาจเป็น default)

เกิดขึ้นเมื่อ:
  • ไม่มีรูปภาพ Active สำหรับ Hotspot ที่ระบุ
  • hotspot_name ไม่ตรงกับที่มีในระบบ
  • ไม่มี Default background (hotspot_name ไม่ระบุ)
ตัวอย่าง Fetch Request
// ตัวอย่างการเรียกใช้ API จาก MikroTik login.html fetch('{{ request.scheme }}://{{ request.get_host }}{% url 'login-background' %}?hotspot_name=hotspot_lib') .then(response => response.json()) .then(data => { if (data.success) { // ตั้ง background document.body.style.backgroundImage = `url(${data.imageUrl})`; document.body.style.backgroundSize = 'cover'; document.body.style.backgroundPosition = 'center'; document.body.style.backgroundRepeat = 'no-repeat'; } else { // ใช้ background สำรอง document.body.style.background = '#1a2535'; } }) .catch(err => console.error('LibLogin API error:', err));
